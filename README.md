# LongStory (롱스토리) 🚀

> **"우리의 이야기가 길어질수록, 이 링크도 자라납니다."**  
> 서버 없이 오직 URL만으로 이어지는, 가장 가볍고 영원한 교환일기 서비스.

<img src="public/favicon.svg" width="200" height="200">


## ✨ 주요 기능

### 1. 🔗 Serverless & Database-less
- 별도의 회원가입이나 로그인, 서버 저장이 필요 없습니다.
- 모든 대화 기록은 **압축된 URL** 안에 안전하게 저장됩니다.
- 링크를 잃어버리지 않는 한, 우리의 이야기는 영원히 남습니다.

### 2. 🎭 멀티 페르소나 모드
누구와 대화하느냐에 따라 맞춤형 질문과 테마를 제공합니다.
- **❤️ 연인 모드**: 설렘 가득한 커플 문답
- **🤜🤛 친구 모드**: 우정을 확인하는 찐친 문답
- **🏡 가족 모드**: 따뜻한 가족 간의 소통
- **🎉 모임 모드**: 단체/회식 분위기를 띄우는 아이스브레이킹

### 3. 📱 모바일 퍼스트 디자인
- 카카오톡, 메시지로 공유하기 최적화된 UI.
- 엄지손가락으로 편하게 조작할 수 있는 **Thumb Zone** 설계.
- **Glassmorphism** 디자인으로 세련되고 몰입감 있는 경험 제공.

### 4. 👨‍👩‍👧‍👦 그룹 릴레이 기능
- **닉네임 시스템**: 누가 쓴 답변인지 타임라인에 표시됩니다.
- **스마트 공유**: "오늘의 질문" 텍스트가 함께 복사되어 단톡방 공유가 쉽습니다.

## 🛠 기술 스택

- **Frontend**: React, Vite
- **Styling**: TailwindCSS, Framer Motion (애니메이션)
- **Logic**: LZ-String (URL 압축 알고리즘)
- **Deploy**: GitHub Pages

## 🚀 시작하기

### 설치 및 실행
```bash
# 프로젝트 클론
git clone https://github.com/juhee77/question-history.git

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 배포
```bash
npm run deploy
```
